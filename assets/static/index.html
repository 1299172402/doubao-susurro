🚀快速开始

  1. 打开 豆包网页版，登录并进入一个对话
  2. 按 F12 打开开发者工具 → Network / 网络 标签
  3. 点击网页中的任意一个对话 → 在 Network / 网络 中 寻找最新一条 single（https://www.doubao.com/im/chain/single）请求
  4. 右键该请求 → Copy / 复制 → Copy as cURL (Bash) / 复制为 cURL (Bash)
  5. 在打开的浏览器页面中，粘贴从豆包复制的 cURL 内容
  6. 点击「💾 保存配置」，然后点击「🚀 获取消息」测试是否正常

🎤使用方法

  1. 在手机上对着豆包的同一个对话说话
  2. 按 Ctrl + V 粘贴到任何输入框中，享受语音输入的便利！
