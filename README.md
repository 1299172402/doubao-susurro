# [Doubao Susurro](https://github.com/1299172402/Doubao-Susurro) – 开源跨设备输入工具

开源跨设备输入工具，依托手机豆包应用即可使用，无需额外安装程序。手机书写、口述的文字可一键直传电脑任意输入框，顺畅衔接日常办公与创作流程。

## 🎯 适用场景

- **代码辅助输入**：适配 OpenCode、Claude Code 等命令行编程工具，接入语音快速输入。
- **低噪轻声录入**：办公室、图书馆、夜间宿舍等安静场景，对着手机小声说话即可识别，不会出声打扰他人。
- **无线手写替代**：无需专属软件与驱动，手机直接充当手写板，手写内容快速同步电脑端。
- **长辈简易录入**：无需掌握拼音，也不用鼠标操作电脑自带手写功能，上手就能轻松完成文字输入。

## ✨ 核心亮点

- **高精度语音转文字**：支持嘈杂环境、自动添加标点符号、中英文混合输入。
- **实时同步**：自动写入系统剪贴板，需要时按 <kbd>Ctrl</kbd>+<kbd>V</kbd> 即可粘贴。
- **自动输入模式**：启用后，光标定位到输入框即可直接填入文字，省去手动粘贴。
- **后台托盘运行**：静默不打扰，随时可用。
- **网页可视化配置**：无需手动修改配置文件。
- **静默启动 & 开机自启**：支持 Windows / Linux / macOS。

### 🚀 快速上手

1.  **获取配置文件**：
    * 打开 [豆包网页版](https://www.doubao.com)，登录并进入一个对话。
    * 按 <kbd>F12</kbd> 打开开发者工具 → **Network**（网络）标签。
    * 在对话中发送一条消息，找到最新一条 `single`（`https://www.doubao.com/im/chain/single`）请求。
    * 右键该请求 → **Copy** → **Copy as cURL (Bash)**。

2.  **启动并配置**：
    * 双击运行 `Doubao-Susurro.exe`，浏览器会自动打开配置页面。
    * 将复制的 cURL 内容粘贴进去，点击「💾 保存配置」。
    * 点击「🚀 获取消息」测试一下是否正常。

3.  **开始使用**：
    * 现在，在你的手机上，对着豆包的**同一个对话**说话或手写。
    * 回到电脑，打开记事本或任何输入框，按下 <kbd>Ctrl</kbd>+<kbd>V</kbd> 即可看到刚才输入的文字！
    * 对于“自动输入”，只要光标在输入框内，然后对手机豆包说话，文字就会直接显示在输入框中。
