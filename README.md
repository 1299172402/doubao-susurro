# Doubao Input

手机随手手写 / 语音输入，文字实时同步到电脑

不会拼音、打字慢？想手机输入直接上电脑编辑？
本地 AI 工具语音不好用、识别不清晰？

**Doubao Input** 是一个让你在手机上**手写或语音**输入，文字就能**自动输入到电脑**的小工具。它会自动把你在手机上说/写的文字粘贴到电脑上当前打开的任意输入框，无缝衔接你的工作流。

> 说明：开源独立工具，非豆包官方输入法，专注「**手机输入 → 电脑实时接收**」跨设备联动。

## ✨ 核心适用场景
- 🎤 **AI 工具语音增强**
  
  支持命令行输入，给 DeepSeek、Copilot 等做外接高精度语音输入，嘈杂环境、轻声说话也能精准识别中英混合内容。

- ✍️ **撰稿 / 网文 / 创作专用**
  
  不用守着键盘打字，随时随地手机口述剧情、文稿大纲、灵感碎片，语音说完直接同步到电脑，码字更顺畅、思路不中断。

- 🖊️ **免驱动无线手写板**
  
  手机手写代替电脑手写板，无需装驱动，手写内容直接同步到电脑。

- 👴 **长辈简易电脑输入**
  
  不会拼音也能用电脑，手机手写/语音，直接上电脑打字，零学习成本。

## 🎯 功能特性
- 高精度语音转文字，支持嘈杂环境、标点、中英混输
- 实时同步，自动写入系统剪贴板
- 可选**自动焦点输入**，无需手动粘贴
- 后台托盘运行，静默不打扰
- 网页可视化配置，不用改配置文件
- 低延迟实时消息轮询

## 🚀 快速上手

1.  **获取配置文件**：
    *   在电脑上打开 [豆包网页版](https://www.doubao.com)，**进入一个具体的对话**（重要！）。
    *   按 <kbd>F12</kbd> → **Network** → 随便发一条消息。
    *   找到 `single` 请求 → 右键 → **Copy** → **Copy as cURL (Bash)**。

2.  **启动并配置**：
    *   双击运行 `doubao-input.exe`，浏览器会自动打开配置页面。
    *   将复制的 cURL 内容粘贴进去，点击「💾 保存配置」。
    *   点击「🚀 获取消息」测试一下是否正常。

3.  **开始使用**：
    *   现在，在你的手机上，对着豆包的**同一个对话**说话或手写。
    *   回到电脑，打开记事本或任何输入框，按下 <kbd>Ctrl</kbd>+<kbd>V</kbd> 即可看到刚才输入的文字！如果开启了“自动输入”，则连粘贴这一步都省了。


### 💡 Windows 开机自启
如果想让工具在电脑启动时自动运行，可以创建一个 VB 脚本以静默模式启动：

1.  在工具目录下新建一个 `doubao-input-start.vbs` 文件，内容如下：
    ```vbs
    Dim ws
    Set ws = Wscript.CreateObject("Wscript.Shell")
    ws.run "doubao-input-app.exe -silent", vbhide
    Wscript.quit
    ```

2.  为这个 `.vbs` 文件创建快捷方式，然后把它放到系统的启动文件夹里：
    ```
    %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup
    ```

现在，每次开机它都会安静地在后台运行，随时待命。
